Show and hide elements using ngIf and hidden in Angular2

ngIf is a structural directive, it creates or destroy contents inside the DOM conditionally. While hidden directive simply show and hide the HTML DOM conditionally.

What are structural directives?

Structural directives are responsible for HTML layout. They shape or reshape the DOM's structure, typically by adding, removing, or manipulating elements.

But There are few differences between ngIf and hidden, they are

  • ngIf is a structural directive, it creates or destroy content inside the HTML DOM conditionally. if the expression is false then the corresponding HTML element and it's content won't be created.
  • While hidden directive just show or hide the content with css, i.e. adding/removing element by adding css property display: none/block;. It's HTML and contents are always created irrespective to condition but they are not visible if the expression is false.

ngIf else

The most common usage of the ngIf directive is to conditionally show the inline template as seen in this example:


<button (click)="show = !show">{{show ? 'hide' : 'show'}}</button>
show = {{show}}
<div *ngIf="show">Text to show</div>

Like other languages Angular also provies the if else capabilities and if it is necessary to display a template when the expression is false use the else template binding.

Note that the else binding points to a <ng-template> labeled #elseBlock. The template can be defined anywhere in the component view but is typically placed right after ngIf for readability. See the example below.


<button (click)="show = !show">{{show ? 'hide' : 'show'}}</button>
show = {{show}}
<div *ngIf="show; else elseBlock">Text to show</div>
<ng-template #elseBlock>Alternate text while primary text is hidden</ng-template>

[hidden]

The Angular [hidden] directive is used to show and hide the HTML DOM. See the example below.


<button (click)="hidden = !hidden">{{hidden ? 'show' : 'hide'}}</button>
hidden = {{hidden}}
<div [hidden]="hidden">Text to show</div>
